# 🚀 Data Structures Studio 2.0

Современный интерактивный учебный проект на Python с GUI-окном, полным GUI-интерфейсом на Tkinter.

В проекте реализованы:

- 📚 Стек
- 🧾 Очередь
- 🌳 Бинарное дерево поиска
- 🕸 Граф

---

## ✨ Идея проекта

Этот проект показывает, как можно изучать классические структуры данных не только через консоль, 
но и через удобное графическое окно.

Пользователь работает с каждой структурой данных через отдельную вкладку:
- вводит значения;
- нажимает кнопки;
- сразу видит результат;
- наблюдает состояние структуры в реальном времени.

---

## 🎨 Почему проект выглядит современно

Проект оформлен как аккуратное мини-приложение:

- темная стильная тема;
- вкладки для разных структур данных;
- удобные карточки с управлением;
- кнопки и поля ввода;
- живое обновление данных прямо в окне;
- понятный интерфейс даже для новичка.

---

## 🧠 Что реализовано

### Стек

- добавление элемента;
- удаление верхнего элемента;
- просмотр верхнего элемента;
- очистка;
- отображение размера и текущего состояния.

### Очередь

- добавление элемента;
- удаление первого элемента;
- просмотр первого элемента;
- очистка;
- отображение размера и текущего состояния.

### Бинарное дерево поиска

- вставка чисел;
- поиск числа;
- симметричный обход;
- прямой обход;
- обратный обход;
- вычисление высоты;
- текстовая визуализация дерева.

### Граф

- направленный и ненаправленный режим;
- добавление ребер;
- хранение в виде списка смежности;
- очистка графа;
- обход в ширину BFS;
- отображение всех связей.

## ▶️ Как запустить

- Через PyCharm: Открой файл main.py и нажми Run.
- Через Windows: Дважды нажми на run.bat.
